True Organic Hub

Project Overview

True Organic Hub is an advanced e-commerce platform designed to promote and sell organic products. This project includes a robust backend implementation using Laravel, integrating RESTful APIs secured with Laravel Passport for authentication. The platform supports features like product management, category association, and dynamic search functionality, ensuring scalability and maintainability with service-layer architecture.

Features

  • Product Management: Create, read, update, and delete products with categories and image uploads.
  • API Integration: RESTful APIs for external integrations and API testing.
  • Secure Authentication: Token-based authentication using Laravel Passport.
  • Dynamic Search: Search products by name or description.
  • Clean Architecture: Service-layer architecture for reusable and maintainable code.
  • API Testing: Endpoints tested with tools like Postman for reliability.

Setup Instructions

Prerequisites

  • PHP (>= 8.0)
  • Composer
  • Node.js and npm
  • MySQL or any supported database
  • Laravel (latest version installed via Composer)

1. Clone the Repository

git clone https://github.com/your-repository/true-organic-hub.git
cd true-organic-hub

2. Install Dependencies

composer install
npm install

3. Setup Environment Variables

  1. Copy .env.example to .env: 
    cp .env.example .env
  2. Update the .env file with your database credentials: 
    DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=your_database_name
DB_USERNAME=your_database_user
DB_PASSWORD=your_database_password

4. Generate Application Key

php artisan key:generate

5. Migrate and Seed the Database

php artisan migrate
php artisan db:seed

6. Install Passport

php artisan passport:install

Add the Laravel\Passport\HasApiTokens trait to your User model to enable API authentication.

7. Serve the Application

php artisan serve

Visit http://127.0.0.1:8000 in your browser.

API Endpoints

Authentication

  • Login: /api/login
  • Register: /api/register

Product Management

  • Get All Products: GET /api/products
  • Get Product by ID: GET /api/products/{id}
  • Create Product: POST /api/products (Protected)
  • Update Product: PUT /api/products/{id} (Protected)
  • Delete Product: DELETE /api/products/{id} (Protected)

Testing

1. Postman

Import the provided Postman collection (if available) and test all API endpoints.

2. Automated Testing

php artisan test
